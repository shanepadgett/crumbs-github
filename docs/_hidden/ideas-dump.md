# Ideas Dump

This file is a scratchpad for raw ideas and rambling thoughts.

## Process

1. Dump thoughts here (speech-to-text, quick notes, etc.)
2. Run the `convert-ideas` prompt to extract these into mini-PRDs in `docs/ideas/`
3. The prompt will clear this file after processing.

## The Dump

- extension to dynamically hide all other extensions, prompts, etc so agents stays focused on targeted extension folder we tell them to investigate
